## 🚀 Getting Started

### **🔧 Installation**
To run the project locally, follow these steps:

1. **Clone the repository**  
   ```bash
   git clone https://github.com/wobiR99/lendsqr-fe-test.git
   cd lendsqr-fe-test
2. **Install dependencies **
    ```bash
    npm install
3. **Start the development server**
    ```bash
    npm run dev
